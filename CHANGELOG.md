# 更新记录

## v1.2.3 - 2026-06-11

- 调整 OTA 更新地址：更新检测接口改为通过 `https://ghfast.top/` 代理访问 GitHub Release 的 `update.json`。
- 调整 OTA 安装包地址：发布用 `update.json` 中的 `apkUrl` 同样使用 `https://ghfast.top/` 代理头。

## v1.2.2 - 2026-06-10

- 优化首页分页体验：遥控器向下移动到“加载更多”按钮时自动加载下一页，无需再按确认键。

## v1.2.1 - 2026-06-10

- 强化 OTA 更新解析：兼容带 UTF-8 BOM 的 `update.json`，避免部分 CDN 或上传工具添加 BOM 后导致更新检查失败。

## v1.2.0 - 2026-06-10

- 新增 OTA 更新第一版：启动后读取 GitHub Release 附件 `update.json`，发现更高 `versionCode` 时提示“有新版本”。
- 支持应用内下载更新包：下载 GitHub Release 中的 APK，下载完成后可跳转系统安装器安装。
- 增加安装包 SHA-256 校验：`update.json` 提供校验值时，下载完成后会校验 APK 完整性。
- 优化更新体验：支持“立即更新 / 稍后再说 / 重新下载 / 安装更新”状态，并在缺少未知应用安装权限时跳转系统授权页。
- 优化首页顶部按钮焦点：`刷新(1)`、`历史(2)`、`搜索(3)`、`直播(4)` 聚焦时显示更明显的高亮、边框和阴影。

## v1.1.0 - 2026-06-09

- 新增电视直播：主页新增 `直播(4)`，从 `https://nav.zhenxx.de5.net/iptv.txt` 获取频道列表并默认播放第一个频道。
- 直播支持遥控器切台：左右键循环切换频道，确认/播放暂停键弹出左侧半透明频道列表，上下键立即切台。
- 直播支持数字选台：数字键可输入频道号，超出频道数时提示“无频道内容”。
- 优化直播观感：左上角频道信息与播放错误提示会在 2 秒后自动隐藏，减少遮挡。
- 支持 HTTP 直播流：允许播放 IPTV 源中的 `http://...m3u8` 地址。

## v1.0.1 - 2026-06-06

- 优化正式版体积：启用 R8 代码压缩和资源裁剪，移除 release 中不需要的 Compose 预览依赖。
- release APK 从约 `30.97 MB` 降至约 `4.23 MB`。

## v1.0.0 - 2026-06-06

- 发布首个正式版：版本号升级为 `1.0.0`，版本码升级为 `10000`。
- 新增正式版打包配置：release 签名从本机 `local.properties` 或环境变量读取，密钥不进入仓库。
- 产出已签名 release APK，可用于电视盒子或 Android 设备侧载安装。

- 修复电视盒子和 Android 手机桌面不显示应用的问题：同时声明 TV `LEANBACK_LAUNCHER` 与普通 `LAUNCHER` 入口，并将 Leanback 特性改为非强制。
- 优化详情页播放入口：首屏新增“立即播放”，低高度设备使用紧凑布局，避免播放源和剧集列表被简介挤出可视区域。
- 新增观看历史：记录影片 ID、名称、封面、播放线路、具体集数、播放地址、播放进度、总时长和更新时间。
- 播放器实时保存历史：播放开始、播放过程中每 5 秒、退出播放器时保存观看进度。
- 首页顶部新增“观看历史”快捷入口，并提供完整历史页，可直接从历史恢复到对应线路、集数和进度继续播放。
- 简化首页历史入口：移除首页顶部的观看历史横排模块，仅保留“历史”按钮，观看记录统一在历史页查看。
- 优化播放体验：播放器底部“上一集 / 下一集 / 返回详情”控制栏会在无操作 4 秒后自动隐藏，遥控操作时重新显示，播放错误时保持显示。
- 新增播放线路：在如意线路之外加入“鸭鸭”和“量子”资源站；鸭鸭和量子均支持备用接口回退。
- 优化多线路历史：观看历史记录内容线路信息，继续播放时会回到对应线路、集数和播放进度。
- 调整线路展示位置：首页固定使用如意数据源；进入详情后，在“播放线路”中聚合同名影片的如意、鸭鸭、量子可播放线路。
- 新增内容过滤：过滤名称、分类、标签、简介或历史记录中包含“伦理”的影视资源。
- 扩展内容过滤：继续过滤“电影解说”相关分类、影片和历史记录；线路恢复为仅保留“如意”一线。
- 优化播放器体验：新增倍速切换、播放结束自动跳下一集，并在返回详情页时记忆当前选集焦点。
- 优化首页分类：支持一级/二级分类展示，点击一级分类时聚合加载其下所有子分类影片。
- 修复首页分类：启动和刷新首页时主动请求分类接口，避免影片列表接口不返回分类时只显示“全部”。
- 调整首页默认内容：首页默认保持“全部”选中，但影片列表优先展示“国产剧”内容。
- 新增播放线路：量子作为首页默认线路，并保留如意作为第二条播放线路。
- 优化量子分类：过滤演员、新闻资讯及其子分类，首页只保留影视相关分类。
- 新增首页快捷键：数字 1 刷新、数字 2 打开历史、数字 3 打开搜索。
- 优化遥控器快捷键：返回首页时焦点恢复到“全部”，播放界面数字 1/3 可切换上一集/下一集。
